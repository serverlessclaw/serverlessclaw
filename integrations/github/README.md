# GitHub Integration

The GitHub Integration repository for ServerlessClaw.

## Structure

- `src/agents`: Specialized agents for GitHub issue resolution.
- `src/adapters`: GitHub-specific adapters for system integration.
- `templates/gh-actions`: CI/CD templates for synchronization.
